# Dependencies List (just for the record)

| package-name | Intro                                                        |
| ------------ | ------------------------------------------------------------ |
| pkg-dir      | Find the root directory of a Node.js project or npm package  |
| npminstall   | Install a package into specific directory                    |
| fs-extra     | fs extra module, provide mkdirp cp rm etc.                   |
| semver       | Compare version for package                                  |
| url-join     | Join all arguments together and normalize the resulting URL. |
| npmlog       | The logger util that npm uses.                               |
